# 单例模式
创建型设计模式,一个类仅能创建一个实例，且对该实例具有全局访问

饿汉模式: 立即实例

懒汉模式：需要时实例，但在多线程场景下会引出线程安全问题

区分: 实例对象的时刻

构造函数: 默认构造函数、拷贝构造函数、拷贝构造函数运算符重载、移动构造函数、移动构造函数运算符重载

## 懒汉模式
解决多线程访问时的线程安全问题

- **双重判断加锁定**
  
  仅在首次获取时存在多次加锁解锁操作
  
  ```
  static taskQueue* get_instance() {
        if (t_Q == nullptr) {
            m_mutex.lock();
            if (t_Q == nullptr) {
                t_Q = new taskQueue;
            }
            m_mutex.unlock();
        }

        return t_Q;
    }
  ```
